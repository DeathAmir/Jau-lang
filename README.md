# Syntax

### Hello World

```
^This Is A Simple Code^

name = "World"

print("Hello " + name)
```

### Variables

```
^Variables^

user = "DeathAmir"
age = 20

print(user)
print(age)
```

### Functions

```
^Function^

func greet(name) {
    print("Hello " + name)
}

greet("Jau")
```

### Conditions

```
^Condition^

age = 18

if age >= 18 {
    print("Access Granted")
} else {
    print("Access Denied")
}
```

### Loops

```
^Loop^

for i in 0..5 {
    print(i)
}
```

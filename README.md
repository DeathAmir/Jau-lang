# Jau Programming Language

<img src="https://raw.githubusercontent.com/simple-icons/simple-icons/develop/icons/code.svg" width="22"/> **Jau** is a modern programming language focused on simplicity, speed, and developer productivity.

It is designed to stay minimal while still offering powerful capabilities for building modern software.

# Syntax

### Hello World

```
^This Is A Simple Code^

name = "World"

print("Hello " + name)
```

### Variables

```
^Variables^

user = "DeathAmir"
age = 20

print(user)
print(age)
```

### Functions

```
^Function Example^

func greet(name) {
    print("Hello " + name)
}

greet("Jau")
```

### Conditions

```
^Condition Example^

age = 18

if age >= 18 {
    print("Access Granted")
} else {
    print("Access Denied")
}
```

### Loops

```
^Loop Example^

for i in 0..5 {
    print(i)
}
```

# Language Features

| Feature | Description |
|------|-------------|
| Clean Syntax | Easy to read and write |
| Fast Compilation | Lightweight compiler |
| JUR Runtime | Secure and optimized runtime |
| Cross Platform | Runs on Windows, Linux, macOS |
| Memory Safe | Runtime managed memory |
| Modern Toolchain | Compiler + Package Manager |

# زبان برنامه‌نویسی Jau

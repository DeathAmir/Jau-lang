# 🚀 Jau

**Jau** is a modern programming language designed for speed, simplicity, and developer power.

It focuses on clean syntax, strong performance, and a developer‑friendly ecosystem.

---

# ⚡ Core Philosophy

• Simple syntax  
• Powerful runtime  
• Cross‑platform execution  
• Fast compilation  
• Modern tooling  

---

# 🧠 Runtime

Programs run using **JUR (Jau Runtime)**

```
Jau Source
     ↓
Jau Compiler
     ↓
Jau Bytecode
     ↓
JUR Runtime
     ↓
Operating System
```

# 🧩 Syntax

```
^This Is A Simple Code^

user :: "DeathAmir"

print -> "Hello " + user
```

```
^Function Example^

greet(name) =>
{
    print -> "Hello " + name
}

greet("Jau")
```

```
^Condition Example^

age :: 18

age >= 18 ?
{
    print -> "Access Granted"
}
:
{
    print -> "Access Denied"
}
```

```
^Loop Example^

i :: 0

loop i < 5
{
    print -> i
    i++
}
